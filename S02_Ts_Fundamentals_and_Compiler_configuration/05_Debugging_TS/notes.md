# Debugging Ts in Browser

- As we know browser doesn't understand TS so in order to debug TS we need to enable `sourceMap` in config file and ts would generate a source map file of the ts code. That source map file would be used to debug the TS Code.

* Once the Source map file is generated browser will directly open that file and we can debug the TS code in Browser as well.

* Another way is to debug in vscode using Node js.
